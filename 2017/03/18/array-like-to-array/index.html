前言

我们都知道，在 JavaScript 中，函数的参数 arguments 对象只是一个 类数组（Array-like） 对象，而不是真正的数组。那么如果我们想把它变成一个真正的数组对象，该怎么做呢？

下面就以 arguments 为例，总结几种类数组对象转换为数组对象的办法。
